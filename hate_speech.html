Hate Term Count Results

This visualization takes the ADL hate terms founds in, "https://www.adl.org/resources/hate-symbols/search". The words are extracted from the website using Beautiful Soup and stored in a CSV file.

We then use the CSV file and the Pixstory dataframe to count how many times each hate word was counted in the users narratives. Our result count of hate terms is then converted to a JSON file which when is placed into the D3 Tree Map visualization!

This visualization provides insight to how important it is to run Detoxify along with hate term flagging. The top 5 most counted hate terms from the ADL website are as follows: 12, 18, 14, 13, and 28. At a glance it can be difficult to determine if these narratives containing these hate words are actually hateful. It is important to note that the hate flagging method our group created only checks if the word is in the narrative. Thus, it can result in inaccuracies of determining if a narrative is in fact hateful.

According to the ADL website, the number, 12, is a numeric symbol for a racist prison gang. Furthermore, the ADL websites describes what these numbers stand for from a "hateful" perspective, but some users can include these "hate terms" without their post being hateful.

This graph was generated to be compared along with the Detoxify Pie Chart. We can conclude that because the count of hate terms is significantly low compared to the amount of narratives in dataset, and the fact that the Detoxify Pie chart has low hate scores, Pixstory lives up to their mission of keeping a clean platform.
