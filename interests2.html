Interest Metrics

For the Pixstory interest, we are using Circle Packing as visual representation of the number of times a certain interest is being mentioned across all 95,000 Pixstory posts. Circular packing allows user to visualize a hierarchic organization. It is an equivalent of a treemap or a dendrogram, where each node of the tree is represented as a circle and its sub-nodes are represented as circles inside of it.

The top 50 most mentioned interests across entire Pixstory dataset are being displayed in circles where a bigger circle represents an interest being mentioned more in Pixstory posts. The team manually assigned a main interest category to each interest and each color represents a main interest category.
